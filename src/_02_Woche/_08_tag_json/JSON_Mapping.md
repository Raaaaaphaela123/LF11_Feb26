Für das JSON-Mapping brauchst du **diese 3 Dateien**:
***
<https://mvnrepository.com/artifact/tools.jackson.core/jackson-core>
***
<https://mvnrepository.com/artifact/tools.jackson.core/jackson-databind>
***
<https://mvnrepository.com/artifact/com.fasterxml.jackson.core/jackson-annotations>
***

| Datei                           | Download |
|---------------------------------|----------|
| `jackson-core-x.x.x.jar`        |          |
| `jackson-databind-x.x.x.jar`    |          |
| `jackson-annotations-x.x.x.jar` |          |

Lade die neuesten Versionen herunter (.jar) (z. B. 2.17.x) und füge sie deinem Projekt hinzu:


**In IntelliJ / Eclipse:**

- Rechtsklick auf dein Projekt
- „Open Module Settings/Project Structure"
- Libraries
- add Java
- Die drei Dateien auswählen
